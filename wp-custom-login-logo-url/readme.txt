== Description ==

This is a simple plugin that allow you to update the logotype and url of the /wp-admin screen.

== Installation ==

1. Upload the entire `wp-custom-login-logo-url` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

Update your logotype in the Apperance > Customize > Site Identity
The URL will update automaticly to your given site url in General > Settings.
